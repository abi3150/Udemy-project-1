# 🎓 Udemy Clone Project (HTML & CSS)

This project is a simple **Udemy-inspired website UI** built using **HTML and CSS**.
It is designed to practice front-end development by recreating a course platform layout.

---

## 🚀 Features

* 🏠 Home Page with course listings
* 🎯 Course cards (title, instructor, price)
* 🔍 Search bar UI
* 📱 Responsive layout (basic)
* 🎨 Clean and modern design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox / Grid)
